# 🚀 AWS Static Website Project

This project demonstrates how to host a simple static website using AWS S3 (Free Tier).

---

## 📌 Project Overview
I created a basic HTML website and deployed it on AWS S3 to make it accessible over the internet.

---

## 🛠️ Tools & Technologies Used
- AWS S3 (Static Website Hosting)
- GitHub (Version Control)
- HTML

---

## ⚙️ Steps Involved
1. Created a simple HTML file (`index.html`)
2. Uploaded the project to GitHub
3. Created an S3 bucket in AWS
4. Uploaded website files to S3
5. Enabled static website hosting
6. Configured public access using bucket policy
7. Accessed the live website using S3 URL

---

## 🌐 Features
- Simple static website
- Hosted on cloud (AWS S3)
- Publicly accessible via URL

---

## 📷 Output
The website displays a basic welcome message hosted using AWS S3.
